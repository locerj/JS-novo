String

Cadeia de caracteres

podemos usar:

" " aspas duplas

exemplo: console.log("Mayk")


' ' aspas simples

exemplo: console.log('Mayk')


` ` crases - template literals ou template strings / permitem multi linhas de caracteres

exemplo: console.log(`
Maik
`)

podemos também colocar expressões e váriaveis com crases usando ${}

exemplo: console.log(`Mayk ${1 + 1}`)
