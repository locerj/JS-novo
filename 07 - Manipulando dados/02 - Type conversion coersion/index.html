Type conversion (typecasting) vs Type coersion

- Alteração de um tipo de dado para outro tipo.

Type coersion:

- quando JS troca o tipo de dado implicidamente.

console.log('9' + 5)

- o resultado será a string 95, já que como estamos concatenando
uma string com um número, o Js tranforma o dado do tipo number 5 no tipo string '5'.

type conversion:

- quando o tipo de dado é trocado explicitamente.

console.log(Number('9') + 5)

- o resultado será o number 14, já que como estamos trocandoo tipo
string para o tipo number e somando, e tranformando o dado do tipo
string '9' no tipo number 9.
