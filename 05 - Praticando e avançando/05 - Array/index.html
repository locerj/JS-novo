Array

Criação de Array

Vazio:

const animals = [];

Com atributos:

const animals = [

'Lion',

'Monkey,

'Cat'

]

Acessando valores de um Array:

Acessando o Array inteiro:

console.log(animals)

Quantidades de itens de um Array:

Podemos saber qtos itens existem em um Array usando .length

ex: animals.length

Acessando apenas um item do Array:

- Para acessar apenas um valor do array é necessário um índice entre colchetes [].

- A contagem de itens de um Arraycomeça com 0(zero).

console.log(animals[1])
