Concatenando e interpolando variaveis

- Multiplos argumentos na função.

- escrita do texto + variável.

- Podemos usar as variáves no texto.

Concatenando:

- para concatenar usamos o sinal de +.

ex: let name, age, isHuman

name = 'Marcelo'

age = 38

isHuman = true

console.log('O ' + name + ' tem ' + age + ' anos.')

Interpolando:

- Usando template literals or template strings.

- para interpolar usamos as crases, e ${} com a variável dentro.

ex: console.log(`O ${name} tem ${age} anos.`)
