Uso de Função

- Usamos funções para criar um bloco de código, que pode ser chamado
e utilizado várias vezes, sem ser necessário repetir o código

Declaração:

function nomeDaFunction() {

console.log("Function")

console.log("Function")

console.log("Function")

}

Chamando função:

nomeDaFunction() {

}
