Function Hoisting

Chamando a função antes da declaração:

- A função sofre um hoisting, ou seja, sua declaração é lida antes da chamada.

- Fazendo com que mesmo sendo chamada andtes de declarada, a função funcione sem problemas.

sayMyName()

function sayMyName() {

console.log('Marcelo')

}

Declarando uma função em uma variável:

- No caso de função anônima, o hoisting não acontence, e a chamada
da função dá erro

sayMyName()

const sayMyName = function() {

console.log('Marcelo')

}
