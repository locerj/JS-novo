Function scope

Criamos uma variável global:

let subject = 'create video'

Criamos uma variável ao criar um parametro na função:

- A variável (subject) criada só funciona no escopo da função.

- Se o parêmetro subject não fosse criado, a função usaria a
variável subject global, alterando assim o seu valor.

function createThink(subject) {

subject = 'study'

return subject

}

- Logo a variavel subject global é diferente da variável da função.

console.log(createThink(subject))

console.log(subject)
