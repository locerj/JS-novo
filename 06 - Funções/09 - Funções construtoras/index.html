Funções construtoras

Function() constructor

- expressõ new

- criar um novo objeto

- this keyword

- um modelo para criar outros objetos

- è uma boa prática começar a função contrutora com letra maiúscula

Declaração:

function Person(name) {

this.name = name

this.walk = () => {

console.log('andando')

}

}

const marcelo = new Person('Marcelo')

console.log(marcelo)

- Podemos criar várias instacias do mesmo objeto

const joao = new Person('João')

console.log(joao)

console.log(marcelo.walk())

Construtores do JS

let nome = String('Marelo')
